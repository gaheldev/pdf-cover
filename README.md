# Run

### From source code

```
python main.py
```

### Compile it yourself

Clone the repo
```
git clone git@github.com:gaheldev/pdf-cover.git
cd pdf-cover
```

Install the dependencies
```
pip install virtualenv
virtualenv .venv
source .venv/bin/activate
./ubuntu-dependencies
```

Build and run the binary
```
make build
./dist/pdfcover
```
